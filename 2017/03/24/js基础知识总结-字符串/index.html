js基础知识总结-字符串

发表于 2017-03-24 | 分类于 前端学习-JavaScript

　　字符串是js中很常见的数据类型，同数组一样，也有许多操作方法，下面就总结一下。

字符方法

charAt()

  • 功能：获取特定位置的字符
  • 返回值：指定位置的字符

  • 语法：Str.charAt(index)
    Str：被操作的字符串
    index：指定的字符的索引值，如果是负数，返回值为空

    // 定义一个字符串
    var str = “skyblueWZZQ/文者字清”;
    // 获取字符
    var result = str.charAt(6);
    // 控制台输出返回值
    console.log(result); // 输出结果为 e

    charCodeAt()

  • 功能：获取特定位置的字符的编码
  • 返回值：指定位置字符的字符编码
  • 语法：Str.charCodeAt()
    Str：被操作的字符串
    idnex：指定字符的索引值，如果是负数，返回值为空
// 定义一个字符串
var str = "skyblueWZZQ/文者字清";
// 获取字符
var result = str.charCodeAt(6);
// 控制台输出返回值
console.log(result);    // 输出结果为 101

操作方法

拼接

concat()

  • 功能：拼接字符串
  • 返回值：拼接后的字符串
  • 语法：Str.concat(str1,str2,str3...)　参数一个或者多个字符串，用逗号间隔
  • 说明：通常情况下，使用 + 更为方便快捷 
// 定义字符串
var str1 = "skyblueWZZQ";
var str2 = "文者字清";
// 拼接
var result1 = str1.concat("/",str2);
// 使用 + 拼接
var result2 = str1 + "/" + str2;
// 控制台输出
console.log(result1);    // 输出结果为  skyblueWZZQ/文者字清
console.log(result2);    // 输出结果为  skyblueWZZQ/文者字清

截取

slice()

  • 功能：截取指定位置字符串
  • 返回值：截取后的新字符串
  • 语法：Str.slice(start,end)
    start：截取起始位置（包含该位置），必选
    end：截取的结束位置（不包含该位置），可选
  • 说明：当传入的参数出现负数时，方法会将字符串长度加上传入的参数作为起始位置或者结束位置进行截取。

    注意：当参入的参数是同向的（都是负数或者都是正数）起始位置始终是小于结束位置的，否则返回值为空

// 定义字符串
var str = "skyblueWZZQ";
//截取字符串 传入一个参数
var result1 = str.slice(2);
//截取字符串 传入两个参数
var result2 = str.slice(2, 5);
//截取字符串 传入一个负数参数
var result3 = str.slice(-2);
//截取字符串 传入两个负数参数
var result4 = str.slice(-5, -2);
// 控制台输出
console.log(result1); // 输出结果：yblueWZZQ
console.log(result2); // 输出结果：ybl
console.log(result3); // 输出结果：zq
console.log(result4); // 输出结果：eWZ

substr()

  • 功能：截取指定位置字符串
  • 返回值：截取后的新字符串
  • 语法：Str.substr(start,end)
    start：截取起始位置（包含该位置）
    end：截取的结束位置（不包含该位置），可选
  • 说明：当传入的参数出现负数时，方法会将字符串长度加上传入的第一个参数作为起始位置进行截取。

    注意：如果第二个参数为负数，则返回空字符串

// 定义字符串
var str = "skyblueWZZQ";
//截取字符串 传入一个参数
var result1 = str.substr(2);
//截取字符串 传入两个参数
var result2 = str.substr(2, 5);
//截取字符串 传入一个负数参数
var result3 = str.substr(-2);
//截取字符串 传入两个负数参数
var result4 = str.substr(-5, -2);
// 控制台输出
console.log(result1); // 输出结果：yblueWZZQ
console.log(result2); // 输出结果：yblue
console.log(result3); // 输出结果：zq
console.log(result4); // 输出结果：空

substring()

  • 功能：截取指定位置字符串
  • 返回值：截取后的新字符串
  • 语法：Str.substr(start,end)
    start：截取起始位置（包含该位置）
    end：截取的结束位置（不包含该位置），可选
  • 说明：当传入的参数出现负数时，方法会将所有负数转换成0；

    注意：该方法还会将参数中小的数字作为起始位置

// 定义字符串
var str = "skyblueWZZQ";
//截取字符串 传入一个参数
var result1 = str.substring(2);
//截取字符串 传入两个参数
var result2 = str.substring(2, 5);
//截取字符串 传入一个负数参数
var result3 = str.substring(-2);
//截取字符串 传入两个负数参数
var result4 = str.substring(-5, -2);
//截取字符串 传入两个参数 第一个参数大于第二个
var result5 = str.substring(5, 2);
// 控制台输出
console.log(result1); // 输出结果：yblueWZZQ
console.log(result2); // 输出结果：ybl
console.log(result3); // 输出结果：skyblueWZZQ
console.log(result4); // 输出结果：空
console.log(result5); // 输出结果：ybl

位置

indexOf()

  • 功能：在指定字符串中从前向后搜寻子字符串
  • 返回值：包含子字符串，则返回该子字符串所在的索引值；不包含，则返回 -1
  • 语法：Str.indexOf(str,index)
    str：要查询的子字符串
    index：指定字符串中的搜索位置，向后查找

    注意：该方法通常用来判断一个字符串中是否包含某个字符串

// 定义字符串
var str = "skyblueWZZQ";
// 查询字符串
var result1 = str.indexOf("W");
// 查询字符串
var result2 = str.indexOf("文");
// 控制台输出
console.log(result1);    // 输出结果：7
console.log(result2);    // 输出结果：-1

lastIndexOf()

  • 功能：在指定字符串中从后向前搜寻子字符串
  • 返回值：包含子字符串，则返回该子字符串所在的索引值；不包含，则返回 -1
  • 语法：Str.lastIndexOf(str,index)
    str：要查询的子字符串
    index：指定字符串中的搜索位置，向前查找

去除空格

trim()

  • 功能：删除字符串前面和后面的所有空格
  • 返回值：新字符串
  • 语法：Str.trim()
    Str：要处理的子字符串

    注意：该方法是ES5新增的，使用时注意兼容性

// 定义字符串
var str = "  skyblue WZZQ  ";
// 去除前后的空格
var result = str.trim();
console.log(result);    // 输出结果：skyblue WZZQ

大小写转换

toLowerCase()

  • 功能：将字符串转换成小写格式
  • 返回值：新字符串
  • 语法：Str.toLowerCase()
    Str：要处理的子字符串 
// 定义字符串
var str = "skyblueWZZQ";
// 转换成小写格式
var result = str.toLowerCase();
console.log(result); // 输出结果：skybluewzzq

toUpperCase()

  • 功能：将字符串转换成大写格式
  • 返回值：新字符串
  • 语法：Str.toUpperCase()
    Str：要处理的子字符串 
// 定义字符串
var str = "skyblueWZZQ";
// 转换成大写格式
var result = str.toUpperCase();
console.log(result); // 输出结果：SKYBLUEWZZQ

匹配方法

match()

  • 功能：匹配符合条件的字符串
  • 返回值：一个数组
  • 语法：Str.match(exp)
    exp：正则表达式或者regexp对象
// 定义字符串
var str = "skyblueWZZQ";
// 匹配
var result = str.match(/b/);
console.log(result);

输出结果：

search()

  • 功能：匹配符合条件的字符串
  • 返回值：返回字符串中第一次出现匹配项的索引值
  • 语法：Str.search(exp)
    exp：正则表达式或者regexp对象
// 定义字符串
var str = "skyblueWZZQ";
// 匹配
var result = str.search(/b/);
console.log(result); // 输出结果：3

replace()

  • 功能：对字符串进行匹配，然后替换匹配到的内容
  • 返回值：替换后的新字符串
  • 语法：Str.replace(a,b)
    a：字符串或者regexp对象
    b：字符串或者函数
// 定义字符串
var str = "skyblueWZZQ";
// 匹配替换
var result = str.replace(/blue/,"文者");
console.log(result); // 输出结果：sky文者WZZQ

split()

  • 功能：基于指定的分隔符将一个字符串分割成多个子字符串，然后将结果放到一个数组中
  • 返回值：由分隔后的子字符串组成的数组
  • 语法：Str.split(a,b)
    a：字符串或者regexp对象
    b：指定返回的数组的长度
// 定义字符串
var str = "sky,blue,W,Z,Z,Q";
// 分隔
var result1 = str.split(",");
var result2 = str.split(",",2);
console.log(result1); // 输出结果：["sky", "blue", "W", "Z", "Z", "Q"]
console.log(result2); // 输出结果：["sky", "blue"]

总结：js中的字符串方法和数组的有些是通用的，如：concat()，indexOf()等，使用时要注意区分，千万不要和数组的方法混淆…^_^
