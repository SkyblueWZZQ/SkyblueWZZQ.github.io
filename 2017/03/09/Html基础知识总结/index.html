title: Html基础知识总结
date: 2017-03-09 18:00:13

tags: 前端学习

学了这么久的前端，现在说起最基础的知识却说不上来几点，索性就来系统的总结一下吧，有不足之处，望各位不吝赐教^_^

html文本结构

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>

<body>
</body>

</html>

这是html5的文本结构

  • <!DOCTYPE html>代表文档遵守的html版本规范，是浏览器识别文档版本的命令。注意：<!DOCTYPE html>不是html标签
  • < html>< /html> 是文档主体，html的所有内容都包含在这对标签内
  • < html lang=”en”>中的 lang=”en” 是设置语言种类的属性
  • < head>< /head>是网页的头部信息，包含着 meta title style 等标签对
  • < meta>< /meta>根据对它的一些属性设置不同的值，可以设置网页的一些基本信息，例如：charset属性，表示网页编码格式。特别是在开发移动端web网页时，meta标签的设置显得尤为重要。
  • < title>< /title>设置网页的标题，是 head 标签中必须要包含的。会显示在浏览器窗口的标题栏或状态栏上。同样，当把文档加入用户的链接列表或者收藏夹或书签列表时，标题将成为该文档链接的默认名称
  • < body>< /body>是文档的内容主体部分，网页显示的内容都包含在body标签对内。
