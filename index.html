创建

// 也可以传递数组内容

html基础知识总结-常用标签

发表于 2017-03-10 | 分类于 前端学习-html

　　html中标签的种类有很多，根据占位方式可以划分为行内元素、行内块元素和块元素。

行内元素

  • 行内元素：元素宽高由内容决定，不能通过样式设定。
    行内元素有：span、a、i、b、em、strong
    　　< span>< /span>：大多使用这个标签包含文本内容
    　　< a href=””>< /a>：定义超链接，href属性值是链接地址，多为目标地址
    　　< i>< /i>：内容显示为斜体
    　　< b>< /b>：内容显示为粗体
    　　< em>< /em>：强调，语气较重，显示为斜体
    　　< strong>< /strong>：强调，比em语气轻一些，显示为粗体
